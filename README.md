# 👛 ארנק דיגיטלי — Digital Wallet

אפליקציית מובייל (Web/PWA) לאחסון והצגת **כרטיסים מגנטיים**, כולל **תעודת זהות** ו**רשיון נהיגה**.

בנוי עם **React + TypeScript + Vite + Supabase**, בעיצוב **Slate & Stone**, עם תמיכה מלאה ב־RTL ועברית.

## יכולות
- 🗂️ אחסון כל סוגי הכרטיסים: תעודת זהות, רשיון נהיגה, מועדון, אשראי, תחבורה, בריאות ועוד.
- 🪪 שדות ייעודיים לכל סוג כרטיס (מספר ת.ז + אימות ספרת ביקורת, דרגות רשיון, תוקף וכו').
- 📷 **צילום כרטיס עם חיתוך אוטומטי מושלם** — צילום **חזית וגב** במצלמה או העלאת תמונה, זיהוי גבולות הכרטיס, כוונון פינות ו**יישור פרספקטיבה** לתמונה ישרה ביחס כרטיס אשראי סטנדרטי. ללא תלות חיצונית, עובד גם ללא רשת.
- 🌗 **מצב כהה** מלא (בהיר / כהה / לפי המערכת) עם שמירת ההעדפה — מתג מהיר בכותרת הארנק והגדרה מלאה בעמוד ההגדרות.
- 📊 הצגת **ברקוד / קוד QR** לסריקה (CODE128 / EAN13 / QR).
- ⭐ סימון כרטיסים מועדפים, חיפוש וסינון לפי סוג.
- 🎨 בחירת צבע לכל כרטיס, תצוגה בסגנון כרטיס מגנטי אמיתי.
- ☁️ אחסון בענן עם Supabase (Auth + RLS) — **או** מצב מקומי (localStorage) ללא הגדרות.

## הרצה מהירה
```bash
cd digital-wallet
npm install
npm run dev
```
פתחו את הכתובת שמופיעה (ברירת מחדל `http://localhost:5173`).

> ללא הגדרת Supabase האפליקציה רצה במצב **מקומי** — הנתונים נשמרים במכשיר, אין צורך בהתחברות. מושלם לבדיקה מהירה.

## חיבור ל־Supabase (אופציונלי, לסנכרון בענן)
1. צרו פרויקט ב־[supabase.com](https://supabase.com).
2. ב־**SQL Editor** הריצו את הקובץ [`supabase-schema.sql`](./supabase-schema.sql) (יוצר טבלת `cards` + RLS).
3. העתיקו `.env.example` ל־`.env.local` ומלאו:
   ```
   VITE_SUPABASE_URL=https://xxxx.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJhbGci...
   ```
4. הריצו מחדש `npm run dev`. כעת נדרשת הרשמה/התחברות, וכל משתמש רואה רק את הכרטיסים שלו.

## מבנה הפרויקט
```
src/
  lib/          שכבת נתונים: types, supabase, storage (ענן/מקומי), validators, imageScan (זיהוי+יישור)
  context/      AuthContext (התחברות), WalletContext (כרטיסים), ThemeContext (מצב כהה)
  components/   CardTile, Barcode, BottomNav, CardScanner (צילום וחיתוך)
  pages/        Login, Wallet, AddCard, CardDetail, Settings
```

### איך עובד החיתוך המושלם?
[`src/lib/imageScan.ts`](./src/lib/imageScan.ts) מזהה את גבולות הכרטיס (היטל אנרגיית שיפוע), והמשתמש יכול לגרור את ארבע הפינות. לאחר מכן מחושב **homography** (פתרון מערכת לינארית 8×8) שממפה את המרובע המעוות למלבן ישר, עם **דגימה בי־לינארית** — כך שכרטיס שצולם בזווית יוצא ישר וחתוך בדיוק. הכל ב־JavaScript טהור, ללא OpenCV או ספריות כבדות.

## אבטחה
- כל הנתונים נשמרים תחת המשתמש המחובר בלבד באמצעות **Row Level Security** של Supabase.
- במצב מקומי הנתונים נשמרים אך ורק ב־`localStorage` של הדפדפן ואינם נשלחים לשום מקום.
