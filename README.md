# 👛 ארנק דיגיטלי — Digital Wallet

אפליקציית מובייל (Web/PWA) לאחסון והצגת **כרטיסים מגנטיים**, כולל **תעודת זהות** ו**רשיון נהיגה**.

בנוי עם **React + TypeScript + Vite**, בעיצוב **Slate & Stone**, עם תמיכה מלאה ב־RTL ועברית.

> **אחסון מקומי בלבד.** כל הנתונים נשמרים במכשיר (בדפדפן) — שום דבר לא נשלח לשום שרת.
> ראו [ADR-0002](./docs/adr/0002-local-only-storage.md) לרקע ההחלטה.

## יכולות
- 🗂️ **8 סוגי כרטיסים:** תעודת זהות, רשיון נהיגה, מועדון, אשראי, גיפט קארד, תחבורה, בריאות ואחר — עם שדות ייעודיים לכל סוג.
- 🎠 **קרוסלה אנכית אינסופית** — גוללים/מחליקים והכרטיסים מסתובבים בתלת-ממד ובלולאה.
- 📷 **צילום כרטיס + חיתוך מושלם** (חזית וגב): מצלמה חיה או העלאה מהגלריה, זיהוי גבולות ויישור פרספקטיבה. ללא תלות חיצונית.
- 🔤 **OCR** (Tesseract.js) ממלא אוטומטית שם/מספר/תאריכים; אימות ספרת ביקורת לת.ז.
- 🔄 **היפוך תלת-ממד** לכרטיס עם חזית וגב (הקשה או החלקה).
- 📊 הצגת **ברקוד / QR**, וגם **סריקת ברקוד חיה** למשיכת מספר הכרטיס.
- 👤 **פרופיל מקומי** (שם + תמונה) המוצג בכותרת.
- 💾 **ייצוא/ייבוא JSON** של הארנק — גיבוי ידני והעברה בין מכשירים.
- 🌗 **מצב כהה** (בהיר/כהה/מערכת), ⭐ מועדפים, 🔍 חיפוש וסינון, 🎨 21 צבעים.

## הרצה
```bash
cd digital-wallet
npm install
npm run dev
```
פתחו את הכתובת שמופיעה. אין צורך בהתחברות, בחשבון או בהגדרות — האפליקציה רצה מיד.

## גיבוי (חשוב)
מכיוון שהאחסון מקומי, **ניקוי הדפדפן או החלפת מכשיר ימחקו את הארנק**. השתמשו ב־
**הגדרות → גיבוי ושחזור → ייצוא גיבוי** כדי לשמור קובץ JSON, ו־**ייבוא גיבוי** כדי
לטעון אותו במכשיר אחר.

## מבנה הפרויקט
```
src/
  lib/          types, storage (localStorage), validators, imageScan (חיתוך), ocr
  context/      WalletContext (כרטיסים), ThemeContext (מצב כהה), ProfileContext (פרופיל)
  components/   CardCarousel, CardTile, CardScanner, BarcodeScanner, FlipCard, Barcode, BottomNav, WalletIcon
  pages/        Wallet, AddCard, CardDetail, Settings
docs/adr/       החלטות ארכיטקטורה (ADR)
CONTEXT.md      גלוסר התחום (ubiquitous language)
```

## פריסה
פרוס כאתר סטטי (למשל Vercel). `vercel.json` כולל rewrite ל־SPA כדי שנתיבים עמוקים
(כמו `/card/:id`) יוגשו כ־`index.html`. אין משתני סביבה — הכל מקומי.
